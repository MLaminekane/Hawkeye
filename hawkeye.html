// How it works
Three commands.
Full visibility.
Initialize

Add to your project

One command creates the config. Zero changes to your code or workflow.

$ npx agentblackbox init
Record

Wrap your agent

Prefix your agent command. AgentBlackBox intercepts everything transparently.

$ agentblackbox record -o "Add JWT auth" -- claude-code
Inspect

See everything

Launch the dashboard. Replay sessions, inspect diffs, track costs, review drift.

$ agentblackbox serve
